Prompt Master

Sistema web completo para gerenciamento e criação de prompts de IA estruturados, oferecendo interface intuitiva para múltiplas categorias de conteúdo.

Manifesto

O Prompt Master tem como missão democratizar e padronizar a criação de prompts de IA, fornecendo uma plataforma robusta e flexível que atende desde usuários iniciantes até profissionais avançados. Nossa visão é ser a ferramenta de referência para gerenciamento de prompts, promovendo eficiência, organização e qualidade na interação com sistemas de IA.

Arquivos
app.py

Arquivo principal da aplicação Flask, contendo todas as rotas, modelos de dados e lógica de negócio para gerenciamento de prompts.

forms.py

Definições dos formulários WTForms para criação e edição de prompts em diferentes categorias.

requirements.txt

Lista de dependências Python necessárias para executar a aplicação.

manage.py

Script para gerenciamento de migrações de banco de dados com Flask-Migrate.

templates/

Diretório contendo todos os templates HTML para as páginas da aplicação.

static/

Arquivos estáticos (CSS, JS, imagens) servidos pela aplicação.

Análise

A arquitetura do Prompt Master é baseada em Flask com SQLAlchemy para persistência de dados. Utiliza WTForms para validação de formulários e Bootstrap para interface responsiva. O sistema suporta múltiplas categorias de prompts com campos específicos, permitindo exportação em formato Markdown. A estrutura modular facilita manutenção e extensibilidade.

Funções

Interface intuitiva para criar prompts em 14 categorias diferentes, com campos específicos para cada tipo de conteúdo.

Sistema completo para visualizar, editar e gerenciar prompts existentes com interface amigável.

Exportação de prompts em formato Markdown estruturado para uso em diferentes plataformas de IA.
Versionamento

Política de versionamento semântico (SemVer). Versão atual: 1.0.0 - Lançamento inicial com suporte completo às 14 categorias de prompts.

FAQ

Clone o repositório, instale as dependências com `pip install -r requirements.txt` e execute `python app.py`.

Suporte a 14 categorias: texto, imagem, áudio, código, vídeo, documento, apresentação, infográfico, animação, website, dados, tradução, resumo e marketing.

Sim, os prompts podem ser exportados em formato Markdown estruturado para fácil uso em diferentes plataformas de IA.
Sugestões

Mantenha o arquivo .env seguro e não o commite no repositório. Use ambientes virtuais Python para isolamento de dependências. Faça backup regular do banco de dados SQLite.
